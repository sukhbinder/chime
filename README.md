# CHIME
chime in command line


# Install
pip install chime

# How to use?

```python
chime
```

# Command line options
```python
chime -h
usage: chime-script.py [-h] [-m {0,15,30}]

Hourly Chimer

optional arguments:
  -h, --help            show this help message and exit
  -m {0,15,30}, --minutes {0,15,30}
                        Time in seconds
```
